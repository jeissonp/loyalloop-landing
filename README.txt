LoyalLoop — Landing Page
========================

Descripción
-----------
Este repositorio contiene la landing page de **LoyalLoop**, un SaaS que digitaliza las tarjetas de fidelidad de negocios locales (cafeterías, barberías, tiendas, etc.) sin necesidad de apps ni registros.  
La página está construida como un único archivo `index.html` con HTML, CSS y JavaScript puros.

Secciones principales
---------------------
- Hero con propuesta de valor y vista previa de la tarjeta digital.
- Cómo funciona (3 pasos sencillos).
- Funciones clave del producto.
- Tabla de precios (plan Gratis y Pro).
- Testimonios de clientes.
- Llamado a la acción final y footer.

Requisitos
----------
- Cualquier navegador moderno.
- No se requieren dependencias ni build: basta con abrir `index.html`.

Uso local
---------
1. Clona el repositorio:
   - `git clone <URL_DEL_REPOSITORIO>`
2. Entra a la carpeta del proyecto:
   - `cd landing`
3. Abre `index.html` en tu navegador:
   - O bien sirve el archivo con un servidor estático si lo prefieres.

Despliegue
----------
Puedes desplegar esta landing fácilmente en:
- GitHub Pages
- Netlify
- Vercel
- Cualquier hosting estático
